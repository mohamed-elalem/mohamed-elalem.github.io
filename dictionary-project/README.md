## Running this project

- Install **cmake**, **docker**, **docker-compose** then run the following commands:
  - `make up`
  - `make import`
- Unzip `./dictionary-project.zip` and import it in netbeans.
- **PHPMyAdmin** is available on `http://localhost:9090`
